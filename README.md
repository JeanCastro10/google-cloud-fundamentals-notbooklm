# google-cloud-fundamentals-notbooklm
Estudos e projetos práticos em Google Cloud Platform, abrangendo infraestrutura (VPC, VMs), segurança (IAM) e arquiteturas Cloud Native e Serverless

Este repositório reúne anotações e atividades práticas sobre o ecossistema do Google Cloud. O conteúdo explora os fundamentos da computação em nuvem sob o modelo de tecnologia como serviço

Tópicos principais incluídos:
Infraestrutura Global: Configuração de VPCs, sub-redes e gestão de Regiões e Zonas para garantir alta disponibilidade e baixa latência
Computação: Implementação de máquinas virtuais via Compute Engine, explorando diferentes famílias de máquinas para otimização de custo e performance
Arquitetura Cloud Native: Desenvolvimento de aplicações stateless e uso de soluções serverless (como Cloud Run) e mensageria assíncrona com Pub/Sub
Segurança e Operações: Aplicação da matriz de responsabilidade compartilhada, gestão de acessos via IAM e monitoramento de infraestrutura
Caso Prático: Resolução de desafios de escalabilidade e latência baseados no cenário real da startup TechNova
